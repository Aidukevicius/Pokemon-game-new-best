# Pokemon Sprites Folder

**PLACEHOLDER**

This folder should contain:
- Pokemon sprite images (PNG format, transparent background)
- Recommended size: 96x96px or 128x128px
- Pixel art style to match SNES aesthetic
- Naming convention: lowercase-pokemon-name.png
  * Example: pikachu.png, charizard.png, bulbasaur.png

You can get sprites from:
- PokéSprite project
- Pokémon Showdown sprites
- Custom pixel art creations
