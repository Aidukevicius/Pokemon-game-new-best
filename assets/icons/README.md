# Extension Icons Folder

**PLACEHOLDER**

This folder should contain Chrome extension icons:

Required sizes:
- icon16.png (16x16px) - Toolbar icon
- icon48.png (48x48px) - Extension management page
- icon128.png (128x128px) - Chrome Web Store listing

Design notes:
- Use a Pokéball or Pokédex design
- Retro/pixel art style to match SNES theme
- High contrast for visibility
- PNG format with transparency
